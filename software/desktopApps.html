Rapid Read

Rapid Read is an open source app, available for all desktop operating systems, that helps you read quicker!
We can read quicker if we remove the need for our eyes to travel from one word to the next to read.
Rapid Read does just that!
Check it out on GitHub

Download Rapid Read
